# Index de la documentation

Ce répertoire centralise les informations de référence du projet.

- `architecture.md` : vue d'ensemble et flux principaux.
- `specs_functionnelles.md` : comportement attendu et parcours utilisateurs.
- `specs_techniques.md` : exigences techniques, dépendances et contraintes non fonctionnelles.
- `conventions.md` : conventions de code, branches et revues.
- `roadmap.md` : jalons planifiés et priorités.
- `decisions.md` : décisions architecturales (ADR).

Chaque évolution doit garder ces documents synchronisés avec le code et les responsabilités des agents.
